# Pricing — YT-DLP GUI

## Summary

| Plan | Price | License |
|------|-------|---------|
| Full application | $0 | MIT — free for personal and commercial use |

## What's included (free)

- Unlimited local downloads via yt-dlp
- Video (up to 4K), audio, subtitles, Instagram, and video clipper
- Auto-install of yt-dlp and ffmpeg
- Six UI accent themes, recents history, custom save paths
- Windows installer from GitHub Releases; macOS/Linux via source build

## What's not included

- No paid tiers, subscriptions, or in-app purchases
- No cloud processing — all downloads run on your machine
- No official support SLA (community support via GitHub Issues)
